# Customer Churn Prediction

A Machine Learning project that predicts whether a telecom customer is likely to churn or stay using Logistic Regression. The model is deployed with Streamlit for an interactive web interface.

## Features
- Customer Churn Prediction
- Churn Probability
- Streamlit Web App
- Logistic Regression Model

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

 ## Machine Learning Workflow

1. Load Dataset
2. Data Cleaning
3. Handle Missing Values
4. Label Encoding
5. Feature Scaling
6. Train-Test Split
7. Train Logistic Regression Model
8. Evaluate Model
9. Save Model
10. Deploy using Streamlit


## Model Performance
- ✅ Accuracy: **82%**
- ✅ ROC-AUC Score: **0.86**
- ✅ Precision: **0.68**
- ✅ Recall: **0.58**
- ✅ F1-Score: **0.62**

## Evaluation Metrics:
- Precision
- Recall
- F1-Score
- Confusion Matrix
- ROC-AUC Score
  
 ## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
